# swoole-wechat
基于Swoole实现微信公众号接口

## Installation
- [PHP install](https://github.com/php/php-src)
- [Swoole extension install](https://github.com/swoole/swoole-src)

## composer 下载依赖库
- composer install --perfer-dist

## nginx配置
- doc/swoole-wechat.conf

## 运行
```
cd swoole-wechat/server
php server.php
```
